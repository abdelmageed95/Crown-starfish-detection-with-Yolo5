
starfish - v3 initial_version
==============================

This dataset was exported via roboflow.ai on March 15, 2022 at 7:49 PM GMT

It includes 432 images.
Starfish are annotated in YOLO v5 PyTorch format.

The following pre-processing was applied to each image:
* Auto-orientation of pixel data (with EXIF-orientation stripping)
* Auto-contrast via histogram equalization

No image augmentation techniques were applied.
